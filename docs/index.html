📊 Data Source

This tool visualizes influenza clade nomenclature from the Influenza Clade Nomenclature GitHub repository.

Current Status: Only HA subclade nomenclature is supported (H3N2, H1N1pdm, B/Victoria).

Updates: Data is fetched from GitHub on each session. Use the "Refresh Data" button to get the latest nomenclature.

How to Use

Navigation

  • Click nodes to view clade details in the inspector panel
  • Ctrl/Cmd + Click to select multiple clades
  • Scroll or drag to pan around the tree
  • Scroll wheel to zoom in/out

Customization

  • Label Size: Adjust text size for clade names
  • Edge Length: Control horizontal spacing between nodes
  • Vertical Spacing: Adjust spacing between clades (increase to prevent overlap)
  • Mutation Font: Change size of mutation labels on edges

Search & Filter

  • Search: Type a clade name (e.g., "C.3") to highlight it
  • Filter: Show only descendants of a specific clade
  • Table: Enable to see sortable, filterable data

💡 Tips

  • Increase Vertical Spacing to 40-50px if mutation labels overlap
  • Use the Clear Selection button to deselect all nodes
  • Download the data table as CSV for further analysis
  • The color legend shows major clade groups
